# AI记账软件

这是一个新建在当前仓库下的 **Android 子项目**，目录名为：`ai记账软件`。

## 目标场景

运行环境：**安卓手机**。

核心流程：

1. 用户在手机上截图。
2. 应用检测到截图事件后，弹出提示框。
3. 用户选择是否进入 App。
4. 进入 App 后，导入刚截图的账单图片。
5. AI 自动整理账单信息（金额、商户、分类、时间、支付方式、备注）。
6. 最后再确认是否保存这一次记账信息。

## 当前原型实现

- 使用 **Jetpack Compose** 构建安卓界面。
- 接入 Android 14+ 的截图检测能力：`registerScreenCaptureCallback()`。
- 声明 `android.permission.DETECT_SCREEN_CAPTURE` 权限。
- 提供“模拟截图事件”按钮，方便开发演示。
- 提供“导入刚截图的账单”入口，用于选择截图图片。
- 提供 AI 整理账单草稿页面。
- 提供最终“是否保存本次记账”的确认弹窗。
- 保存后的账单会进入本地列表展示。

## 重要说明

根据 Android 官方截图检测 API：

- 系统可以通知 App “用户刚刚截图了”。
- **但回调本身不会直接把截图图片内容传给 App。**
- 所以这个原型中，采用的是：
  - 检测到截图 -> 弹窗询问
  - 用户进入 App -> 选择刚截图的图片
  - 再进入 AI 整理与保存流程

这也是更符合隐私约束的设计。
